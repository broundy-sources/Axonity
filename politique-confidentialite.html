Politique de confidentialité

1. Collecte des données personnelles

Nous collectons les données personnelles suivantes :

  • Formulaire de contact : nom, email, message
  • Navigation : données de connexion (adresse IP, navigateur)

2. Finalité du traitement

Vos données sont utilisées pour :

  • Répondre à vos demandes de contact
  • Améliorer notre site web
  • Respecter nos obligations légales

3. Base légale du traitement

Le traitement de vos données personnelles est basé sur :

  • Votre consentement pour les formulaires de contact
  • L'intérêt légitime pour l'amélioration du site

4. Destinataires des données

Vos données peuvent être partagées avec :

  • Notre équipe interne uniquement
  • Nos prestataires techniques (hébergeur) sous contrat de confidentialité
  • Aucune vente ou cession à des tiers

5. Durée de conservation

  • Données de contact : 3 ans maximum
  • Logs de connexion : 12 mois maximum

6. Cookies

Notre site utilise les cookies suivants :

  • Cookies techniques : fonctionnement du site
  • Cookies de performance : amélioration de l'expérience

Vous pouvez gérer vos préférences cookies dans les paramètres de votre navigateur.

7. Vos droits (RGPD)

Vous disposez des droits suivants :

  • Droit d'accès : connaître vos données traitées
  • Droit de rectification : corriger vos données
  • Droit d'effacement : supprimer vos données
  • Droit à la portabilité : récupérer vos données
  • Droit d'opposition : vous opposer au traitement
  • Droit de limitation : limiter le traitement

8. Exercer vos droits

Pour exercer vos droits, contactez-nous :

  • Email : contact@axonity.fr
  • Courrier : 8 allée des hontines 33650 Labrede

Nous nous engageons à répondre dans un délai de 30 jours maximum.

10. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données contre tout accès non autorisé, modification, divulgation ou destruction.

Cette politique peut être modifiée. Les modifications importantes vous seront notifiées.
